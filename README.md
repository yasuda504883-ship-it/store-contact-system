# 店舗連絡ガント管理

店舗への連絡状況を、担当者別・ステータス別に管理するための簡易Webアプリです。

## できること

- 店名を入力して店舗を追加
- エリア、担当者、ステータス、予定日を管理
- 担当者別に絞り込み
- ステータス別に絞り込み
- 進捗をガントチャート風に表示
- ブラウザに自動保存
- iPhoneでも操作しやすいレスポンシブ表示

## 担当者

- 濱治
- 羽賀
- 佐藤
- 鈴木
- 安田

## ステータス

- 未連絡
- 連絡済
- 返信待ち
- 撮影決定
- 完了

## ファイル構成

```text
store-contact-system
├── index.html
├── style.css
├── script.js
└── README.md
```

## 使い方

`index.html` をブラウザで開くだけで使えます。

GitHub Pagesを有効にすると、スマホからURLで開けるようになります。
